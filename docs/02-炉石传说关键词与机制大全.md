## 一、随从战斗类关键词

### 1.1 嘲讽 (Taunt)

**规则定义**：
- 具有嘲讽的随从在战场上时，对手**必须**优先攻击此随从
- 如果场上同时存在多个嘲讽随从，对手可**任意选择**其中一个攻击
- 嘲讽只限制**攻击**，不影响指向性法术和英雄技能（对方仍可对非嘲讽随从施法）

**特殊交互**：
| 场景 | 结果 |
|------|------|
| 两个嘲讽随从同时在场上 | 对手可任选一个攻击 |
| 嘲讽 + 潜行 | 潜行的嘲讽随从**不可被攻击**，对手优先攻击其他嘲讽随从；若无其他嘲讽，则无视此随从可攻击英雄 |
| 嘲讽 + 免疫 | 免疫的嘲讽随从不可被攻击；对手若没有其他可攻击的嘲讽随从，可绕过所有嘲讽直接攻击英雄 |
| 嘲讽 + 冻结 | 嘲讽随从被冻结后仍保留嘲讽效果，对手仍受其限制（但可攻击其他嘲讽随从） |

**程序实现要点**：
```
攻击合法性检查伪代码：
  tauntMinions = 敌方场上所有具有嘲讽且非免疫的随从
  if (tauntMinions.length > 0):
    攻击目标必须是 tauntMinions 中的一员
  else:
    攻击目标可以是任何敌方随从或敌方英雄
```

### 1.2 冲锋 (Charge)

**规则定义**：
- 具有冲锋的随从**不受召唤失调影响**，被打出的当回合即可攻击
- 冲锋随从攻击时需遵守所有常规攻击限制（嘲讽、冻结等）

**与相关概念的区分**：
| 概念 | 当回合可攻击英雄？ | 当回合可攻击随从？ |
|------|:---:|:---:|
| **冲锋** | ✅ | ✅ |
| **突袭** | ❌ | ✅ |
| **召唤失调** | ❌ | ❌ |
| **风怒** | 允许攻击 2 次，但仍受召唤失调限制 | |

**程序实现**：
```
随从攻击状态标记：
  summoningSickness: boolean       // 召唤失调
  canAttackHero: boolean           // 能否攻击英雄（冲锋 = true, 突袭 = false）
  
  canAttack(): boolean {
    if (this.summoningSickness) return false
    // ...其他检查（冻结、攻击次数等）
  }
```

---

### 1.3 突袭 (Rush)

**规则定义**：
- 具有突袭的随从不受召唤失调影响，被打出的当回合**可以攻击敌方随从**
- 但是**不能攻击敌方英雄**（在打出的当回合）
- 从下个己方回合开始，突袭随从恢复正常（可以攻击英雄）

**边界情况**：
- 突袭随从在打出当回合可以攻击任何敌方随从（包括嘲讽随从、潜行随从——潜行失效后）
- 如果突袭随从获得冲锋（如被 buff），则它同时拥有冲锋和突袭，按冲锋规则执行（可以攻击英雄）
- 如果对手场上没有随从，突袭随从打出当回合**无事可做**（不能攻击英雄）
- 敌方满场嘲讽随从时，突袭随从必须攻击嘲讽随从（遵守嘲讽规则）

**程序判断**：
```
minion.charge = true → 当回合可攻击英雄
minion.rush = true → 当回合仅可攻击随从

// 在第二个己方回合时，两者无区别（都可攻击英雄）
```

### 1.4 潜行 (Stealth)

**规则定义**：
- 具有潜行的随从**不可成为敌方指定的目标**
  - 不可被攻击
  - 不可被指向性法术指定（如暗言术：灭）
  - 不可被指向性英雄技能指定（如法师火焰冲击）
- 潜行随从**仍受 AOE 影响**（如烈焰风暴、乱斗）
- 潜行随从**仍可攻击**，攻击后潜行**消失**

**潜行状态的自动消失**：
- 随从发起攻击 → 潜行消失（攻击结算前潜行已消失）
- 随从造成伤害（非攻击方式，如拉格纳罗斯）→ 潜行**不消失**
- 潜行随从被 buff 赋予嘲讽 → 此时嘲讽被压制（无法攻击此随从），潜行结束后嘲讽生效

**潜行 + 嘲讽的特殊规则**：
- 潜行中的嘲讽随从：嘲讽效果被**暂时压制**
- 潜行结束后，嘲讽效果**自动恢复**
- 对手无法攻击潜行中的嘲讽随从，可绕过它直接攻击其他目标

### 1.5 圣盾 (Divine Shield)

**规则定义**：
- 具有圣盾的单位，**第一次受到的伤害降为 0**，然后圣盾消失
- 圣盾只吸收**伤害**，不阻挡以下效果：
  - 直接消灭（如"消灭一个随从"）
  - 属性置换（如"将一个随从的生命值变为 1"）
  - 沉默
  - 变形

**圣盾精确结算**：
```
圣盾随从受到 N 点伤害：
  1. 圣盾检查：存在圣盾？
  2. 是 → 伤害降为 0，圣盾移除
  3. 否 → 正常扣血 N 点

过量伤害与圣盾：
  圣盾吸收全部伤害，无过量概念
  造成 0 点伤害仍会消耗圣盾
```

**圣盾的叠加**：
- 圣盾不可叠加为一个单位上的多个护盾层
- 给已有圣盾的单位再施加圣盾 → 无效果（仍只有一层）
- 圣盾被移除后可以重新获得

---

### 1.6 风怒 (Windfury)

**规则定义**：
- 具有风怒的单位每回合可以攻击 **2 次**
- 每次攻击独立检查合法性（嘲讽、冻结等）
- 适用于随从和英雄（通过武器）

**攻击状态机**：
```
初始状态：可攻击（次数 = 0/2）

第一次攻击 → 状态变为：已攻击 1 次（次数 = 1/2），仍可继续操作
第二次攻击 → 状态变为：已攻击 2 次（次数 = 2/2），不可再攻击

回合结束 → 重置到初始状态
```

**风怒 + 其他关键词交互**：
| 组合 | 行为 |
|------|------|
| 风怒 + 冲锋 | 打出当回合即可攻击 2 次 |
| 风怒 + 冰冻 | 第一次攻击后，若被冰冻，第二次攻击无法进行 |
| 风怒 + 超级风怒(Mega-Windfury) | 覆盖：每回合攻击 4 次 |
| 风怒 + 武器 | 英雄可攻击 2 次，每次消耗 1 点耐久 |

### 1.7 剧毒 (Poisonous)

**规则定义**：
- 具有剧毒的随从造成的**任何伤害**都会消灭目标随从
- 关键：「任何伤害」包括 **0 点伤害**

**剧毒精确结算**：
```
剧毒随从 攻击 目标随从：
  1. 剧毒随从对目标造成 X 点伤害
  2. 检查：目标是否受到了来自剧毒随从的伤害？
  3. 是 → 目标被消灭（无视剩余生命值）
  4. 目标受到 X 点伤害 → 触发"受到伤害时"效果 → 目标被消灭

特殊：剧毒的生效条件是"造成了伤害"，而非"攻击了"
  - 剧毒随从被误导攻击友方 → 消灭友方
  - 剧毒随从的亡语造成伤害 → 不触发剧毒（来源不是随从本身）
```

**剧毒 + 圣盾交互（重要）**：
```
流程：
  剧毒随从 A 攻击 圣盾随从 B
  → A 对 B 造成 A 的攻击力伤害
  → B 的圣盾吸收伤害，伤害降为 0
  → B 受到了 0 点伤害（来自 A）
  → 剧毒触发：A 造成了伤害（虽然是 0 点）→ B 被消灭
  → B 被消灭，不进行反击
  → A 不受伤 ✓
```

**剧毒 + AOE**：
- 带有剧毒的随从施放 AOE（如通过效果）→ 对所有受伤的随从触发剧毒
- 乱斗类不造成伤害的效果 → 不触发剧毒

### 1.8 吸血 (Lifesteal)

**规则定义**：
- 具有吸血的单位造成伤害时，**同时**为其控制者的英雄恢复等量的生命值
- 吸血的治疗量与造成的伤害量**相等**

**吸血精确规则**：
```
吸血随从造成 N 点伤害：
  → 目标受到 N 点伤害
  → 英雄恢复 N 点生命值
  → 二者同时发生

过量伤害与吸血：
  目标只有 1 血，造成 5 点伤害
  → 目标受到 5 点伤害（4 点过量）
  → 英雄恢复 5 点生命值（过量伤害也计算吸血量）
```

**吸血与法术**：
- 法术也可以具有吸血关键词
- AOE 法术 + 吸血 = 对每个目标独立计算吸血量，英雄恢复总量 = 所有目标受伤之和
- 吸血不限于战斗伤害——任何来源的伤害都可吸血

**吸血 + 护甲**：
- 吸血只恢复生命值，不增加护甲
- 如果英雄已满血（30/30），吸血不产生任何效果（不会转化为护甲）

### 1.9 冻结 (Freeze)

**规则定义**：
- 被冻结的角色**失去其下一次攻击机会**
- 冻结状态持续到**下一个己方回合开始**时自动解除

**冻结的精确时序**：
```
本回合中角色被冻结：
  情况 A：角色本回合尚未攻击
    → 本回合不能攻击
    → 下回合开始，冻结解除，可以正常攻击
  
  情况 B：角色本回合已经攻击过
    → 本回合不受影响（已攻击过）
    → 下回合开始，冻结检查：角色被冻结
    → 下回合不能攻击
    → 再下回合开始，冻结解除

  情况 C：角色已在冻结状态，再次被冻结
    → 无叠加效果（不能「跳过两次攻击」）
    → 冻结状态不变，仍在下个己方回合开始时解除
```

**冻结与冲锋/突袭**：
- 冲锋随从打出后被冻结 → 当回合不能攻击（失去冲锋优势）
- 突袭随从不受冻结影响其「可攻击性」——如果被冻结，当回合同样不能攻击

**冻结的特殊情况**：
- 冻结不影响随从的其他功能（仍可触发回合开始/结束效果、仍提供光环效果、仍可被 buff）
- 冻结不影响随从的嘲讽效果
- 冻结的随从**可以成为攻击目标**

---

### 1.10 免疫 (Immune)

**规则定义**：
- 具有免疫的角色：
  - 不能成为攻击目标
  - 不能成为指向性法术/英雄技能的目标
  - 不受到任何伤害（包括 AOE 伤害、随机伤害）
  - **不被消灭效果影响**（部分卡牌的消灭效果也无法指定免疫单位）
- 但如果某个效果不「指定目标」也不是「伤害」，而是「全局效果」，免疫单位仍可能受影响

**免疫的精确边界**：
| 效果类型 | 免疫单位是否受影响？ |
|----------|:---:|
| 指向性伤害（如火球术） | ❌ 不可指定 |
| AOE 伤害（如烈焰风暴） | ❌ 不受伤害 |
| 随机伤害（如奥术飞弹） | ❌ 不会被随机选中 |
| 直接消灭（如刺杀） | ❌ 不可指定 |
| 全局消灭（如扭曲虚空） | ✅ **会被消灭** |
| 属性设置（如"变为 3/3"） | 取决于实现（通常不可指定） |
| 沉默 | ❌ 不可指定 |
| 自身 buff / 英雄技能 buff | ✅ 仍可 buff |

**特殊情况**：
- 具有免疫的嘲讽随从不阻挡攻击（见 1.1 节）
- 免疫通常是**临时**的（持续到回合结束）
- 永久免疫极为罕见（如某种冒险模式 Boss）

## 二、触发类关键词

### 2.1 战吼 (Battlecry)

**规则定义**：
- 战吼是从**手牌打出**随从（或英雄牌）时触发的效果
- 触发时机：随从入场后、召唤失调赋予前、场上其他"打出后"效果触发前
- 战吼在随从**真正进入战场后**才生效（先入场，再战吼）

**战吼不是召唤触发**：
- "召唤"（Summon）不触发战吼
- 例如：先祖之魂复活随从 → 不触发战吼
- 例如：精神控制技师拉过来的随从 → 不触发战吼
- **打出**（Play）≠ **召唤**（Summon）

**战吼的目标选择**：
- 需要指定目标的战吼：打出随从后弹出目标选择界面
- 不需要目标的战吼：随从入场后自动触发
- 如果战吼指定了目标，且目标在战吼结算前消失（如被其他效果移除），战吼**仍然会触发**，但部分效果可能落空

**战吼结算顺序**：
```
打出随从：
  1. 法力水晶扣除
  2. 随从进入战场
  3. 战吼触发并结算
  4. "打出随从后"效果触发
  5. 召唤失调赋予
```

**战吼 + 伤害的先死后死问题**：
- 如果战吼造成伤害导致场上随从死亡，这些随从的亡语在**战吼完全结算完毕后**才触发

---

### 2.2 亡语 (Deathrattle)

**规则定义**：
- 亡语是随从**死亡时**触发的效果
- 死亡的定义：随从生命值降到 ≤ 0 并从战场移除
- 触发时机：在死亡结算阶段（见 01 号文档 5.4 节）

**亡语的精确规则**：
- 亡语触发时，该随从**已不在战场上**，其提供的所有光环效果已消失
- 亡语的来源（控制者）= 随从死亡时的控制者
- 如果随从被**沉默后死亡**，亡语不触发
- 如果随从被**变形后死亡**，亡语不触发（因为被变形的随从失去了原卡牌的所有效果）
- 如果随从**被洗入牌库后**（不是死亡），亡语不触发

**多个亡语的触发顺序**：
```
多个随从同时死亡：
  → 亡语按这些随从的"入场时间"顺序依次触发
  → 入场早的先触发亡语
  → 注意：是入场时间，不是死亡时间

亡语过程中又导致死亡：
  → 新死亡的随从的亡语在当前亡语结算完毕后触发
  → 嵌套结算（递归）
```

**亡语的所属权**：
- 亡语效果属于随从的当前控制者
- 如果随从被偷取后死亡，亡语效果由**偷取者**触发（新控制者）

**亡语与复活**：
- 被沉默后死亡的随从 → 仍可被复活（在坟场中）→ 复活后沉默效果消失，恢复原始属性
- 被变形后死亡的随从 → 不可被复活（在坟场中是变形后的形态）

---

### 2.3 连击 (Combo)

**规则定义**：
- 具有连击的卡牌，如果**本回合中已经打出过其他任意卡牌**，则在打出时触发额外的连击效果
- 连击是一个**条件触发**：满足条件 → 额外效果生效；不满足条件 → 仅基础效果

**连击计数器**：
```
每位玩家独立维护一个连击计数器：
  回合开始时 → 计数器归零
  每打出一张牌 → 计数器 +1

具有连击的卡牌打出时：
  if (计数器 ≥ 1):
    触发连击效果
  else:
    仅触发基础效果

注意："本回合"包括对手的回合（虽然通常只在己方回合打出连击牌）
```

**连击的触发细节**：
- 幸运币也计为一次打出（可用于触发连击）
- 对手回合中，如果你通过奥秘或效果打出了牌，也会更新连击计数器
- 连击效果是附加在基础效果之上的，不是替代

---

### 2.4 过载 (Overload)

**规则定义**：
- 具有「过载：(X)」的卡牌在打出后，会在**下回合**锁定 X 颗法力水晶
- 过载在 01 号文档 2.3 节有详细描述，此处仅补充交互规则

**过载的叠加与清除**：
- 连续两个回合打出过载牌 → 过载量累加
- 过载没有「清除」机制，只能自然过期（一回合后自动解除）
- 如果被锁定的水晶数量 > 下回合可用水晶数量，下回合**无可用法力水晶**（过载锁定了全部）

**过载与卡牌复制**：
- 如果你的过载牌被对手复制并打出，对手下回合受到过载影响
- 如果你是后手，通过卡牌效果在对手回合打出了过载牌，受到过载影响的是**对手的下回合**（因为现在是对手的回合）

---

### 2.5 激励 (Inspire)

**规则定义**：
- 具有激励的随从，在你使用英雄技能后触发其效果
- 触发时机：英雄技能使用并**完全生效后**

**激励的精确时序**：
```
使用英雄技能：
  1. 检查费用（2 法力水晶）
  2. 检查使用次数（每回合 1 次）
  3. 英雄技能效果结算
  4. "使用英雄技能后"触发
  5. 场上所有具有激励的随从按入场顺序依次触发
```

**激励的交互**：
- 如果英雄技能被替换（如打出英雄牌），激励仍然对新英雄技能有效
- 如果英雄技能的费用被减少至 0，使用 0 费技能仍然触发激励
- 随从被沉默 → 失去激励
- 多个激励随从在场 → 按入场顺序依次触发

---

### 2.6 法术迸发 (Spellburst)

**规则定义**：
- 具有法术迸发的随从，在你**施放一个法术后**触发一次性效果
- **一次性**：每个随从的法术迸发只能触发一次

**法术迸发的精确规则**：
```
施放法术后：
  → 场上所有具有"未触发"法术迸发的友方随从
  → 按入场顺序依次触发其法术迸发效果
  → 触发后，该随从的法术迸发标记为"已触发"
  → 获得视觉上的标记（如灰色边框）

沉默 → 法术迸发标记被移除，可以重新触发
随从返回手牌再打出 → 标记重置
```

**法术迸发与法术类型的限制**：
- 英雄技能不是法术，不触发法术迸发
- 幸运币是法术，会触发法术迸发
- 奥秘被触发（非主动施放）不触发法术迸发

---

### 2.7 暴怒 (Frenzy)

**规则定义**：
- 具有暴怒的随从，在**第一次受到伤害并存活后**触发一次性效果
- 一次性：每个随从的暴怒只能触发一次

**暴怒的精确触发条件**：
```
必同时满足三个条件：
  1. 随从受到了伤害（伤害量 ≥ 1）
  2. 随从在受到伤害后仍然存活（生命值 > 0）
  3. 暴怒尚未触发过

如果随从受到伤害后死亡 → 不触发暴怒
如果随从受到 0 点伤害（如圣盾吸收）→ 不触发暴怒
```

**暴怒的触发时机**：
- 在伤害结算完成后、死亡检查前触发
- 如果暴怒效果造成伤害导致其他随从死亡，按正常死亡结算流程处理

**暴怒的重置**：
- 沉默 → 暴怒标记移除，可以重新触发
- 随从返回手牌再打出 → 标记重置

---

### 2.8 法术伤害 (Spell Damage)

**规则定义**：
- 具有「法术伤害 +N」的随从在场上时，其控制者施放的**法术造成的伤害**增加 N 点
- 这是一个**光环效果**（Aura），随从在场持续生效

**法术伤害的适用范围**：
| 效果类型 | 是否受法术伤害加成？ |
|----------|:---:|
| 法术牌造成的伤害 | ✅ |
| 英雄技能造成的伤害 | ❌（除非英雄技能明确受法伤加成） |
| 随从战吼造成的伤害 | ❌ |
| 随从亡语造成的伤害 | ❌ |
| 武器造成的伤害 | ❌ |
| AOE 法术（对所有目标） | ✅ 每个目标独立享受加成 |

**法术伤害的叠加**：
- 场上多个法伤随从 → 法伤加成累加
- 法伤随从死亡 → 加成立即消失
- 法伤 +N 和法伤 +M 的随从在场 → 总加成 = N + M

---

## 三、卡牌状态类关键词

### 3.1 沉默 (Silence)

**规则定义**：
- 沉默**移除**目标随从上的所有卡牌文本效果、增益（buff）、减益（debuff）和关键词
- 沉默不改变随从的攻击力和生命值（但移除来源为 buff 的攻/血变化）
- 沉默不改变随从的种族标签

**沉默移除的内容**：

| 移除 | 不移除 |
|------|--------|
| 卡牌描述文本中的所有效果 | 随从的基础攻击力/生命值 |
| 所有已获得的关键词（嘲讽、圣盾等） | 随从的基础种族标签 |
| 所有 buff（+攻击力、+生命值等） | 来自光环的 buff（沉默后光环仍然生效） |
| 所有 debuff（-攻击力、冻结等） | 随从的站位 |
| 亡语 | 随从的入场顺序 |
| 法术迸发/暴怒的「已触发」标记 | — |

**沉默的精确行为**：
```
随从受到沉默：
  1. 移除所有 buff 层（攻击力、生命值恢复到基础值）
  2. 移除所有关键词（包括圣盾、嘲讽、风怒等）
  3. 移除卡牌文本（战吼、亡语等不再生效）
  4. 当前的所有持续效果标记被清除
  5. 光环效果重新评估（光环不受沉默影响，重新应用到沉默后的随从）

沉默后随从的展示：
  → 卡牌文本变灰
  → 随从变为「白板」（无效果、无关键词）
```

**沉默与死亡/复活**：
- 被沉默的随从死亡后仍在坟场中
- 从坟场复活被沉默的随从 → 随从恢复原始属性（沉默效果被清除）
- 因此沉默不影响复活后的随从

**沉默与变形**：
- 变形（如妖术、变羊）不等同于沉默
- 变形改变了随从的基础属性，不可恢复

---

### 3.2 奥秘 (Secret)

**规则定义**：
- 奥秘是挂在英雄头像上的隐藏法术
- 奥秘打出后，对手**看不到**具体内容（只看到问号图标）
- 奥秘在**对手回合**满足条件时自动触发

**奥秘的完整规则**（已在 01 文档 3.2 节部分描述）：

**数量限制**：
- 每位玩家最多同时挂 **5 个奥秘**
- 同名奥秘最多 1 个（不能挂两个相同的奥秘）
- 奥秘的触发条件各不相同（如「对手打出一个随从时」「对手攻击你的英雄时」等）

**触发顺序**：
```
事件发生（如：对手打出一张随从）
  ↓
遍历该玩家挂载的所有奥秘（按挂载顺序）：
  for each 奥秘:
    检查其触发条件是否满足？
      满足 → 触发此奥秘（展示、生效、进入坟场）
      不满足 → 检查下一个奥秘
  注意：每次只触发一个奥秘，即使多个奥秘条件同时满足
  ↓
如果触发了奥秘，重新遍历剩余的奥秘
  （因为触发了一个奥秘可能改变游戏状态，影响后续奥秘的触发条件）
```

**奥秘的触发优先于事件**：
- 如果奥秘的触发条件是「对手打出一张随从时」：
  - 奥秘先触发 → 奥秘效果结算（如消灭该随从）→ 随从的「打出后」效果可能因随从已不存在而无法触发
- 这是奥秘最强大的地方：它可以**打断**对手的操作

**常见奥秘触发时机类型**：
| 触发时机 | 示例 |
|----------|------|
| 对手打出随从时 | 狙击、镜像实体、爆炸符文 |
| 对手施放法术时 | 法术反制、扰咒术 |
| 对手攻击英雄时 | 冰冻陷阱、爆炸陷阱、误导 |
| 对手攻击随从时 | 毒蛇陷阱、捕熊陷阱 |
| 对手回合开始时 | 竞争之魂 |
| 你的英雄受到伤害后 | 复仇 |
| 随从死亡后 | 救赎、复仇 |
| 你的回合开始时（极少） | — |

---

### 3.3 发现 (Discover)

**规则定义**：
- 发现：从随机 **3 张**符合条件的牌中选择 **1 张**加入手牌
- 选项之间互不相同（不会出现相同的选项）

**发现的来源池**：
- 发现的牌来自当前游戏环境的所有可用卡牌（标准模式或狂野模式）
- 发现的牌不受牌库限制（可以找到你没有带的牌）
- 发现的牌通常来自中立 + 当前职业的卡牌池

**发现的随机性**：
- 3 张选项完全随机（从符合条件的牌池中随机抽取）
- 如果有权重加成（如职业牌出现概率高于中立牌），则按加权随机

**发现的费用**：
- 发现的牌的费用不受限制（除非文本说明，如「发现一张 3 费的随从」）
- 发现的牌进入手牌后可以正常使用

---

### 3.4 进化 (Adapt)

**规则定义**：
- 进化：从 **10 种进化选项**中随机选择 **3 种**，玩家选择 1 种应用到目标随从

**10 种进化选项**：
| # | 选项名 | 效果 |
|:---:|------|------|
| 1 | 剧毒 | 获得剧毒 |
| 2 | 圣盾 | 获得圣盾 |
| 3 | 风怒 | 获得风怒 |
| 4 | 潜行（持续到下回合） | 获得潜行，持续到下回合开始 |
| 5 | +3 攻击力 | 获得 +3 攻击力 |
| 6 | +3 生命值 | 获得 +3 生命值 |
| 7 | +1/+1 | 获得 +1/+1 |
| 8 | 亡语：召唤两个 1/1 | 获得亡语：召唤两个 1/1 的植物 |
| 9 | 无法成为法术或英雄技能的目标 | 获得「精灵之魂」效果 |
| 10 | 嘲讽 | 获得嘲讽 |

**进化的规则**：
- 3 个随机选项之间互不相同
- 如果所有 3 个选项都不适用 → 仍可选择一个（即使无效果）
- 进化是永久效果（除非说明「持续到……」）

---

## 四、扩展包关键词

### 4.1 回响 (Echo)

**规则定义**：
- 回响：打出此卡后，在手中添加一张**临时复制**
- 临时复制：本回合结束时从手中消失（不进入坟场，直接消失）
- 回响卡在同一回合内可以无限次数打出（有足够法力水晶的前提下）

**回响卡的临时复制特性**：
- 临时复制在回合结束时**消失**（不是弃牌、不是洗入牌库）
- 临时复制与原始卡牌完全相同（属性、费用均一致）
- 临时复制在手牌中占据正常位置

**回响 + 费用减少交互**：
- 如果回响卡的费用被减至 0，可以无限打出
- 费用减少应用于回响卡的临时复制（因为复制品和原始卡完全相同）

---

### 4.2 磁力 (Magnetic)

**规则定义**：
- 具有磁力的机械随从，打出时可以：
  - **单独打出**：正常入场，作为一个独立随从
  - **磁力融合**：与场上一个友方机械随从融合——将其属性叠加（攻击力 + 攻击力，生命值 + 生命值），关键词合并

**磁力的两个选择**：
```
打出磁力随从时：
  ┌─ 选择 1：单独打出 → 放在空位上，作为一个新的随从
  └─ 选择 2：拖到友方机械随从上 → 融合
       ├─ 攻击力 += 磁力随从的攻击力
       ├─ 生命值 += 磁力随从的生命值
       ├─ 磁力随从的关键词合并到目标（圣盾+圣盾=一层圣盾）
       └─ 磁力随从的卡牌文本效果合并到目标
```

**磁力的限制**：
- 只能与**机械**随从融合
- 如果场上没有友方机械随从，只能单独打出

---

### 4.3 双生法术 (Twinspell)

**规则定义**：
- 双生法术打出后，在手中产生一张**不带双生关键词**的法术复制
- 复制是永久的（不会在回合结束时消失）
- 复制使用后消失，不再产生新的复制

```
双生法术打出：
  → 法术效果结算
  → 在手牌中生成一张复制（不含双生关键词）
  → 复制可以正常使用（永久保留）

复制使用：
  → 法术效果结算
  → 不产生任何复制
  → 结束
```

---

### 4.4 复生 (Reborn)

**规则定义**：
- 具有复生的随从，**第一次死亡后**以 **1 点生命值**复活
- 复活后失去复生关键词
- 复活后的随从保留所有 buff/debuff（除生命值重置为 1）

**复生的精确流程**：
```
复生随从死亡：
  1. 随从生命值 ≤ 0，进入待死亡状态
  2. 死亡结算：
     a. 亡语触发（原随从的亡语）
     b. 复生检查：此随从具有复生
     c. 复生触发：此随从以 1 点生命值复活到原位
     d. 复生关键词移除
     e. 复活后的随从保留之前的所有 buff
  3. 复活后的随从视为「新召唤」（召唤失调重新赋予）
```

**复生的视觉指示**：
- 具有复生的随从在卡牌下方有特殊光效（蓝色光环）
- 复活后光效消失

---

### 4.5 祈求 (Invoke)

**规则定义**：
- 祈求是迦拉克隆体系的专用关键词
- 打出祈求牌时：
  - 触发祈求的通用效果：迦拉克隆升级（形态 1 → 2 → 3 → 4）
  - 触发职业独有的祈求效果（如潜行者：随机将一张跟班置入手牌）

**迦拉克隆的升级路径**：
```
初始形态   → 祈求 2 次 → 形态 2
形态 2     → 祈求 2 次 → 形态 3
形态 3     → 祈求 2 次 → 最终形态
最终形态不可再升级

注意：
  - 迦拉克隆不需要在手上/场上也能升级（升级的是「英雄牌」在牌库中的状态）
  - 迦拉克隆在手中会显示当前形态
```

---

### 4.6 流放 (Outcast)

**规则定义**：
- 具有流放的卡牌，在打出时如果处于**手牌的最左端或最右端**，触发额外的流放效果
- 流放是打出时的**即时判断**（位置在打出瞬间判定）

**流放的触发条件**：
```
手牌索引：0, 1, 2, ..., N-1

流放卡打出时：
  if (手牌位置 == 0 OR 手牌位置 == N-1):
    触发流放效果
  else:
    不触发流放效果（仅基础效果）

注意：
  - 手牌位置在抽牌、打出牌后会动态变化
  - 玩家可以通过先打出其他牌来「调整」流放卡到手牌两端
```

---

### 4.7 腐蚀 (Corrupt)

**规则定义**：
- 具有腐蚀的卡牌在手中时，如果你打出**一张费用更高的卡牌**，此卡变为「已腐蚀」形态
- 腐蚀是**一次性**的转变（不可逆）
- 已腐蚀的卡牌具有更强的效果（通常是数值提升）

**腐蚀的触发机制**：
```
腐蚀卡在手牌中：
  每当你打出一张牌：
    检查：打出牌的费用 > 腐蚀卡的费用？
      是 → 腐蚀卡变为「已腐蚀」形态（视觉变化、效果提升）
      否 → 无变化

注意：
  - 腐蚀发生的时间点：费用更高的牌打出后
  - 腐蚀是手牌中的状态变化
  - 已腐蚀的卡牌即使被返回手牌再打出，仍保持已腐蚀状态
```

---

### 4.8 可交易 (Tradeable)

**规则定义**：
- 具有「可交易」的卡牌可以**拖入牌库中**，花费 1 点法力水晶，将这张牌与牌库顶的牌交换
- 可交易操作不算「打出」这张牌

**可交易的精确流程**：
```
玩家拖拽可交易卡牌到牌库区域：
  1. 消耗 1 点法力水晶
  2. 此卡洗入牌库（随机位置）
  3. 从牌库顶抽 1 张牌
  4. 此卡的效果不触发（不是"打出"）
  5. 此卡的亡语/战吼等完全不生效
```

**可交易的限制**：
- 可交易操作和打出此牌是**互斥**的
- 如果选择了可交易，就不能打出此牌（反之亦然）
- 可交易操作可以在任何有足够法力水晶的时机进行

### 4.9 荣誉消灭 (Honorable Kill)

**规则定义**：
- 具有「荣誉消灭」的随从/法术，当它**恰好造成等于目标当前生命值的伤害**（不多不少）时，触发额外效果
- 荣誉消灭只对随从目标生效，对英雄不生效

**荣誉消灭的判断条件**：
```
造成 N 点伤害，目标当前生命值 = N：
  → 造成的伤害恰好等于目标生命值
  → 目标被消灭（伤害正好致死）
  → 触发「荣誉消灭」效果

造成 N 点伤害，目标当前生命值 < N（过量伤害）：
  → 目标仍被消灭，但不触发「荣誉消灭」

造成 N 点伤害，目标当前生命值 > N：
  → 目标存活，不触发「荣誉消灭」
```

### 4.10 巨型 (Colossal)

**规则定义**：
- 具有「巨型 +N」的随从在打出时，会额外召唤 N 个附属部件
- 「+N」指的是附属部件的数量（如巨型 +1 = 1 个附属部件）

**巨型的精确规则**：
```
打出巨型随从：
  1. 巨型随从本体正常入场
  2. 其附属部件自动召唤到巨型随从两侧
  3. 附属部件计入随从数量上限（7 个）
  4. 如果战场不够容纳所有附属部件，则尽可能召唤

附属部件特性：
  - 附属部件有自己的攻击力、生命值（通常在巨型随从的卡牌描述中写明）
  - 附属部件可以独立被攻击、被消灭
  - 附属部件死亡后不触发任何关联效果（除非卡牌描述说明）
```

### 4.11 注能 (Infuse)

**规则定义**：
- 具有「注能(N)」的卡牌在手牌中时，每当友方随从死亡，注能计数器 +1
- 当注能计数器达到 N 时，卡牌变为「已注能」形态（增强版）
- 注能只发生在手牌中

**注能的详细规则**：
```
注能卡在手牌中：
  友方随从死亡事件 → 检查手牌中的注能卡
    → 注能计数器 +1
    → 检查是否达到 N？
      是 → 卡牌变为「已注能」形态
      否 → 继续等待

注能计数器的生命周期：
  - 卡牌被打出 → 计数器重置
  - 卡牌返回手牌 → 计数器保留
  - 卡牌被弃掉 → 计数器丢失（不可恢复）
```

---

### 4.12 快枪 (Quickdraw)

**规则定义**：
- 具有快枪的卡牌，在**抽到此牌的同一回合**打出时，触发额外效果
- 快枪是一个打出时的条件判断

**快枪的判断逻辑**：
```
每张牌维护一个 turnDrawn 属性（被抽到的回合编号）

快枪卡打出时：
  if (当前回合编号 == turnDrawn):
    触发快枪效果
  else:
    不触发快枪效果

注意：
  - 快枪效果在打出时触发，而非抽到时触发
  - 如果快枪卡在手牌中停留超过 1 回合，快枪效果永久失效
  - 如果快枪卡被洗回牌库重新抽到，turnDrawn 重置为当前回合
```

3. **嘲讽 + 免疫**：免疫的嘲讽随从不限制对手攻击

5. **复生 + 冲锋**：复活后仍是新随从，冲锋关键词保留所以可以继续攻击
6. **复生 + 突袭**：复活后的随从没有突袭的优势（召唤时赋予的临时突袭已过期），但如果突袭是关键词则保留
