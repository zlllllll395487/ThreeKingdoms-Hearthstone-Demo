# 三国炉石 · 基础卡牌设计表 v2.0（M0 第一期）

> **本版本核心改动**（v1.0 → v2.0）：
> - 卡池从 55 张 → **40 张精选**
> - 删除超模卡（吕布 8/8、苦肉计 1 费抽 3、司马懿亡语精控等共 8 张）
> - 删除"羁绊"相关设计（桃园结义改为无条件触发）
> - 删除弓兵/方士相关卡（M1 再加入）
> - 兵种简化为 3 种（步兵/骑兵/谋士）
> - 关键词命名修正（突阵→冲锋、先登→突袭、威慑→淬毒、夺气→饮血、疑兵→困阵）
> - **每张卡都附带费用计算说明和与 v1.0 的差异说明**
>
> 配套设计文档：[03-三国炉石核心玩法策划案-v2.md](03-三国炉石核心玩法策划案-v2.md)

## 二、蜀国阵营（9 张）

> **风格**：铺场 + 群体 buff + 步兵协同。配合英雄技能「仁德」（召唤 1/1 义兵）滚雪球。

### 2.1 传说（1 张）

#### S01 · 关羽

| 项 | 值 |
|---|---|
| 费用 | 6 |
| 攻 / 血 | 5 / 6 |
| 类型 | 武将 · 骑兵 |
| 稀有度 | 传说 |
| 关键词 | 突袭 |
| 效果 | **威名**：装备一把「青龙偃月刀」（3/3） |

**设计意图**：关羽武圣，单骑冲阵+巨刀。突袭让他打出当回合就能交换大随从，战吼装备的青龙偃月刀（见兵器章节）让英雄也能输出。

**费用计算**：基准 6 费 = 14 总属性。5/6 = 11，扣 3 因为带突袭（-1 攻）+ 强力威名（-2）= 准确平衡。

**v1.0 → v2.0**：v1.0 没有关羽实体卡（只在英雄候选中提及）。v2.0 把蜀国传说从赵云改为关羽，因为关羽更有代表性，且赵云的"四关键词"在 v1.0 严重超模。

### 2.2 史诗（1 张）

#### S02 · 张飞

| 项 | 值 |
|---|---|
| 费用 | 4 |
| 攻 / 血 | 4 / 5 |
| 类型 | 武将 · 步兵 |
| 稀有度 | 史诗 |
| 关键词 | 镇守 |
| 效果 | **遗志**：对所有敌方武将造成 1 点伤害 |

**设计意图**：当阳桥一声吼退曹军（镇守）。死前 AOE 1 伤模拟猛吼震慑。

**费用计算**：4 费基准 10 总属性，4/5 = 9 + 镇守（0 扣减，因攻 ≥ 4）+ 弱遗志 AOE 1（-1）= 准确。

**v1.0 → v2.0**：v1.0 是 4 费 5/3 + 镇守 + 遗志 AOE 2，5/3 镇守过于"嘲讽墙+反伤"双重收益且 AOE 2 是核武级清场。v2.0 改为 4/5 更耐打 + AOE 降为 1 点平衡。

### 2.3 稀有（3 张）

#### S03 · 桃园结义

| 项 | 值 |
|---|---|
| 费用 | 3 |
| 类型 | 计策 |
| 稀有度 | 稀有 |
| 效果 | 召唤三个 1/2 的「义兵」（步兵） |

**设计意图**：兄弟同心，三个 1/2 比 1/1 更耐打，与「仁德」技能召唤的 1/1 形成层次。

**v1.0 → v2.0**：v1.0 是 3 费 3 个 1/1 + 羁绊触发改为 3 个 2/2 + 铁壁。**砍掉羁绊**（无法触发）+ 提升基础到 1/2（更耐 AOE）。

## 三、魏国阵营（9 张）

> **风格**：兵器 + 护甲 + 控制。配合英雄技能「奸雄」（+2 护甲）打长线消耗。

### 3.1 传说（1 张）

#### W01 · 司马懿

| 项 | 值 |
|---|---|
| 费用 | 6 |
| 攻 / 血 | 5 / 6 |
| 类型 | 武将 · 谋士 |
| 稀有度 | 传说 |
| 关键词 | — |
| 效果 | **威名**：复制对手手牌中一张随机牌 |

**设计意图**：三马同槽，司马懿的核心是"窃取"——但只偷 1 张避免破坏游戏节奏。

**费用计算**：6 费基准 14。5/6 = 11 + 威名复制 1 张牌（≈ 3 费效果）= 14。准确。

**v1.0 → v2.0**：v1.0 是 7 费 5/7 战吼偷 3 张牌库顶 + 亡语精神控制。**精神控制是 10 费法术**，做成传说亡语严重超模；偷 3 张牌也是顶级抽牌。v2.0 改为 6 费偷 1 张手牌，删除亡语精控。

## 四、吴国阵营（9 张）

> **风格**：过牌 + 计策 + 灵活。配合英雄技能「制衡」（抽 1 张）滚动信息差。

### 4.1 传说（1 张）

#### U01 · 周瑜

| 项 | 值 |
|---|---|
| 费用 | 5 |
| 攻 / 血 | 3 / 5 |
| 类型 | 武将 · 谋士 |
| 稀有度 | 传说 |
| 关键词 | — |
| 效果 | **谋略 +2**。在你的回合结束时，摸 1 张牌 |

**设计意图**：周瑜——能让计策伤害 +2（赤壁火攻效果），且持续摸牌（智囊型角色）。

**费用计算**：5 费基准 12。3/5 = 8 + 谋略 +2（≈ 3 费效果）+ 每回合摸 1 牌（≈ 2 费持续效果，但本体易死）= 13。略超 1 点，但 5 费 3/5 身材脆弱平衡了。

**v1.0 → v2.0**：v1.0 是 5 费 3/5 回合结束时随机敌方武将 2 伤 + 摸 1（杀死再摸 1）。**不可控的随机伤害容易爆炸**，且双重摸牌过强。v2.0 改为静态 +2 谋略 + 持续摸 1 牌，效果更可预测、更适合"智囊"主题。

### B. 平衡测试待办

> v2.0 是"基于推理的平衡设计"，必须在实装后做以下测试：

1. **AI 自动对战 100 场**：每个阵营对每个阵营各 100 场，胜率应在 45%-55% 区间
2. **每张卡的"使用率"统计**：使用率 < 5% 的卡需要重做
3. **每张卡的"胜率"统计**：胜率 > 60% 的卡需要削弱
4. **关键词出现频次**：每个关键词在牌组中出现概率应 > 30%（否则删除该关键词或加更多卡）

### C. 接 v1.0 但需要校对的细节

以下设计点是从 v1.0 直接继承但**值得在实装后再校对**：

1. **仁德召唤的 1/1 义兵和 S03 召唤的 1/2 义兵同名**——可能造成混淆，建议改名（如「步卒」vs「义兵」）
2. **关羽战吼装备的青龙偃月刀** vs **从牌组打出的青龙偃月刀**——是同一张兵器卡还是衍生品？规则需明确（建议：是同一张，关羽战吼直接打出此牌而非从手牌）
3. **铁匠的 +1 攻** vs **兵器本身耐久 = 0 销毁**：如果兵器只剩 1 攻 1 耐，铁匠 +1 攻让它变 2 攻 1 耐，仍能攻击吗？规则需明确（建议：可以）
