## 一、视觉风格基调

### 1.1 核心定位

```
精致工笔重彩 × 现代游戏 CG × 史诗气场
```

**参考标杆**：用户提供的刘备立绘参考图（白金主调 + 蜀绿点缀 + 龙纹刺绣 + 楷书风味文字 + 战场旌旗背景）。

**画风分类**：
- ✅ **半写实 CG 插画风**（接近真·三国无双、王者荣耀史诗皮肤精度）
- ❌ 不是水墨工笔（前期方案废弃）
- ❌ 不是 Q 版卡通
- ❌ 不是日漫萌系
- ❌ 不是欧美写实硬派

### 1.2 主色板（从参考图提取）

| 用途 | 色值 | 出处 |
|---|---|---|
| 卡牌主底色 | `#F8F4E9` 米白宣纸 | 参考图衣袍底色 |
| 鎏金高光 | `#D4A017` | 参考图甲胄装饰 |
| 古金描边 | `#B8860B` | 参考图勾边 |
| 朱砂红 | `#C8362F` | 参考图印章 / 旗帜 |
| 墨黑 | `#1A1A1A` | 主文字 / 楷书 |
| 深棕 | `#5C4A3A` | 风味文字 |

> **粗体武将**（关羽 / 张飞 / 黄忠 / 庞统）= 高优先，立绘投入更多。其他武将可降级处理。

### 3.3 设计要求

| 项 | 规格 |
|---|---|
| **画幅尺寸** | **1024×1536 px**（2:3 竖向，固定规格）|
| **格式** | PNG，背景透明 或 半透明氛围背景（参考图就是带氛围背景的）|
| **构图** | **全身或七分身**，角色占画幅 70%，**头部在画面上 1/3**，气势构图（仰视 / 斜对角） |
| **视角** | 多采用**微仰视**，让角色显得高大威严（参考图就是仰视） |
| **背景** | **可带战场氛围背景**（旌旗、远景士兵剪影、云雾），但不要喧宾夺主 |
| **风格** | 半写实 CG 插画，**精致工笔重彩**，画面饱满 |
| **线条** | 中粗墨色勾边 + 数字绘画上色，金线高亮 |
| **配色** | 以**阵营色为主**，搭配**金 / 白 / 朱红** |
| **细节标准** | **服饰必须有刺绣纹理**（龙纹 / 凤纹 / 云纹）；**金属甲胄必须有高光反光**；**人物面部表情必须有神** |
| **风味元素** | 角色右下或左下可加竖排楷书人物简介（仿参考图）+ 朱砂印章 |

### 3.4 风格参考与避免

```
推荐参考：
- 用户提供的刘备立绘参考图（核心基调）
- 真·三国无双 8 / 9 角色立绘
- 王者荣耀 三国系列史诗皮肤（如关羽-黑虎咆哮、貂蝉-仲夏夜之梦）
- 三国志 14 角色卡面
- 网易《三国如龙传》卡牌立绘

避免：
- 日系动漫脸（如《一骑当千》）
- 写实硬派（如《全战三国》）
- 国产手游硅胶塑料感（如劣质三国手游）
- 任何"性感擦边"风格
- 极简水墨（前期方案废弃）
- Q 版卡通
```

## 六、卡牌底框（P0：6 个）

### 6.1 用途说明

卡牌底框是"画框"，决定卡牌视觉档次。

### 6.2 demo 必需清单

| 文件名 | 稀有度 | 适用类型 | 数量 |
|---|---|---|:---:|
| frame_minion_common | 普通 | 武将 | 1 |
| frame_minion_rare | 稀有 | 武将 | 1 |
| frame_minion_epic | 史诗 | 武将 | 1 |
| frame_minion_legendary | 传说 | 武将 | 1 |
| frame_spell | 通用 | 计策（demo 4 张计策不分稀有度）| 1 |
| frame_weapon | 通用 | 兵器（demo 仅 1 张）| 1 |

### 6.3 设计要求

| 项 | 规格 |
|---|---|
| **尺寸** | 单张卡 360×504px（5:7 比例）；@2x 720×1008px |
| **格式** | PNG，带透明（中间立绘窗口透明）|
| **结构** | 上方留 240×280px 区域给立绘窗口；中部留费用 / 攻击 / 生命数值位；下方留 1-3 行文字位 + 风味文字位 |
| **风格** | 精致工笔重彩，**纸张质感 + 鎏金边框 + 龙纹刺绣**（参考图整体氛围） |

### 6.4 各稀有度差异化设计

#### 普通（common）

- 边框色：`#8C8C8C` 青灰
- 装饰：简单回纹边框
- 背景：纯米白宣纸
- 印章：无

#### 稀有（rare）

- 边框色：`#3D7BB8` 青蓝 + 古金线
- 装饰：祥云纹（边框上下两端）
- 背景：米白 + 极淡云纹底图
- 印章：右下角小型阵营印章

#### 史诗（epic）

- 边框色：`#7B3DA8` 紫水晶 + 古金线
- 装饰：玉璧（左右两侧）+ 双层边框
- 背景：米白 + 中等强度云纹底图
- 印章：左下 + 右下双印章

#### 传说（legendary）

- 边框色：`#D4A017` 鎏金 + 朱砂红
- 装饰：**完整龙纹绕框 + 角落鎏金祥云 + 双侧楷书风味文字位**（呼应参考图）
- 背景：米白 + 浮雕龙纹底图
- 印章：**朱砂大印（角落 80×80px）**+ 阵营印章
- **特殊**：选中态有金光波动动画（CSS 实现）

frame_minion_legendary.png
frame_minion_legendary_active.png    传说卡选中态
frame_spell.png
