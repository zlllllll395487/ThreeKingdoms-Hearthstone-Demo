## 一、设计总纲

### 1.1 设计理念

```
    炉石传说核心骨架（100%保留）
          +
    三国世界观皮肤（主题包装）
          +
    三国专属新机制（差异化亮点）
          ↓
    三国炉石  —— 上手零门槛（炉石玩家秒懂），策略有新深度（三国机制加分）
```

### 1.2 保留 vs 改动总览

| 系统 | 策略 | 说明 |
|------|------|------|
| 回合结构 | **完全保留** | 回合开始→主要阶段→回合结束 |
| 法力水晶系统 | **完全保留** | 1-10 颗递增，过载机制重命名保留 |
| 卡牌类型体系 | **保留 + 扩展** | 随从→武将、法术→计策、武器→兵器、英雄牌→名将牌 |
| 攻击与伤害结算 | **完全保留** | 伤害同时结算、死亡结算等核心算法不变 |
| 战场规则 | **完全保留** | 7 随从上限、站位固定 |
| 对局流程 | **完全保留** | 换牌、抽牌、疲劳等全不变 |
| 关键词系统 | **保留 + 重命名 + 扩展** | 炉石关键词用三国术语重命名，新增 4 个三国专属关键词 |
| 英雄技能 | **重新设计** | 每阵营不同技能，贴合三国角色 |
| 阵营系统 | **新增** | 替代炉石的职业系统 |

### 1.3 游戏命名

- 中文名：**三国炉石** / **三国战记**（备选）
- 英文名：**Hearth of Three Kingdoms**（备选）

## 二、阵营与英雄系统

### 2.1 四大阵营

炉石的职业体系 → 三国的阵营体系，每个阵营有独特的玩法风格和英雄技能。

| 阵营 | 领袖 | 称号 | 代表色 | 核心风格 | 炉石玩法类比 |
|------|------|------|:---:|------|------|
| **蜀** | 刘备 | 仁德之主 | 🟢 绿 | 群体增益、召唤部队、回血 | 圣骑士 + 萨满 |
| **魏** | 曹操 | 乱世奸雄 | 🔵 蓝 | 控制、偷牌、资源压制 | 牧师 + 术士 |
| **吴** | 孙权 | 江东猛虎 | 🔴 红 | 过牌、防守、后期爆发 | 德鲁伊 + 战士 |
| **群雄** | 吕布 | 天下无双 | 🟣 紫 | 高攻快攻、单挑、高风险高回报 | 猎人 + 恶魔猎手 |

### 2.2 英雄技能设计

每个阵营的英雄技能费用为 **2 点法力水晶**，每回合限用 1 次。

| 阵营 | 英雄技能名 | 效果 | 设计意图 |
|------|-----------|------|----------|
| **蜀**（刘备） | **仁德** | 使一个友方随从获得 +1/+1 | 强化己方场面，鼓励随从交换后的补刀能力 |
| **魏**（曹操） | **奸雄** | 将对方牌库顶的一张牌复制到你的手牌中 | 资源差打法，前期空过时也能积累优势 |
| **吴**（孙权） | **制衡** | 抽一张牌，如果本回合中你已经打出过牌，额外再抽一张 | 鼓励「先做事再补牌」的节奏 |
| **群雄**（吕布） | **无双** | 本回合中，你的英雄获得 +2 攻击力 | 激进的进攻技能，亲自上阵打出伤害 |

### 2.3 英雄技能设计说明

**蜀 - 仁德 (+1/+1)**：
- 泛用性最强，任何场面都有用
- 不能补刀（不像法师火焰冲击能打 1），但可以帮随从存活和交换
- 鼓励铺场和随从交换的中速打法

**魏 - 奸雄（偷牌库顶）**：
- 没有场面影响，但获得额外资源
- 偷来的牌你不知道是什么，有随机性和惊喜感
- 可能偷到关键卡牌，改变战局走向

**吴 - 制衡（抽牌）**：
- 条件触发额外抽牌，鼓励合理安排出牌顺序
- 和「制衡」典故契合：孙权善于平衡各方势力
- 后期法力充裕时，每回合都能触发额外抽牌

**群雄 - 无双（英雄+2攻）**：
- 最激进的技能，每回合都可以造成 2 点伤害
- 符合吕布天下无双、亲自上阵的角色形象
- 配合武器体系极强，但卖血风险也高

### 2.4 后续可扩展英雄

每个阵营可设计多个可选英雄（类似炉石不同职业的不同英雄皮肤/英雄牌）：

| 阵营 | 可选英雄 |
|------|----------|
| 蜀 | 刘备、诸葛亮、关羽、赵云 |
| 魏 | 曹操、司马懿、张辽、夏侯惇 |
| 吴 | 孙权、周瑜、孙策、陆逊 |
| 群雄 | 吕布、董卓、袁绍、张角 |

不同英雄可设计不同的英雄技能，作为后续扩展内容。

## 三、卡牌类型三国化

### 3.1 类型映射总表

| 炉石原类型 | 三国版类型 | 图标 | 说明 |
|:---:|:---:|:---:|------|
| Minion | **武将** | ⚔️ | 上场的战斗单位，占领战场格子 |
| Spell | **计策** | 📜 | 一次性法术效果 |
| Weapon | **兵器** | 🗡️ | 装备给英雄的武器 |
| Hero Card | **名将牌** | 👑 | 替换英雄和英雄技能 |
| Secret | **锦囊** | 🎭 | 挂在英雄上的隐藏计策 |
| Quest | **霸业** | 🏆 | 完成条件获得奖励的任务牌 |

### 3.2 武将（= 随从）

- 所有炉石随从规则**完全继承**
- 种族体系映射：
  - 武将分为不同**兵种**：步兵、骑兵、弓兵、谋士、方士
  - 兵种之间有协同卡牌效果（如同兵种在场时获得增益）

### 3.3 计策（= 法术）

- 所有炉石法术规则**完全继承**
- 三国化命名风格：
  - 火攻、水淹七军、空城计、苦肉计、连环计、反间计、借东风、草船借箭
- 计策的子类型：
  - **普通计策**：常规法术
  - **锦囊**（= 奥秘）：隐藏的计策牌
  - **霸业**（= 任务）：完成条件获得奖励

### 3.4 兵器（= 武器）

- 所有炉石武器规则**完全继承**
- 代表性兵器：
  - 青龙偃月刀（关羽）、丈八蛇矛（张飞）、方天画戟（吕布）
  - 诸葛连弩（远程武器）、倚天剑（曹操）、古锭刀（孙权）
- 兵器属性：攻击力 / 耐久度，和炉石完全一致

### 3.5 名将牌（= 英雄牌）

- 打出后替换英雄和英雄技能
- 获得护甲值（= 士气值）
- 名将牌示例：
  - 卧龙诸葛亮：战吼→发现一张计策牌，英雄技能→火攻（3 费打 3）
  - 武圣关羽：战吼→装备青龙偃月刀（3/2），英雄技能→威震华夏（2 费，本回合+2 攻）

### 4.2 三国专属新关键词

在炉石关键词基础上，新增 **4 个**三国主题专属关键词：

---

#### 【新关键词 1】单挑（Duel）

**定义**：发起单挑的武将选择一个敌方武将，双方**互相造成等同于自身攻击力的伤害**。此过程不触发「受到攻击时」效果，但触发「受到伤害时」效果。

**规则细节**：
```
单挑流程：
  1. 发起单挑的武将选择一个敌方武将
  2. 发起者对目标造成等于其攻击力的伤害
  3. 目标对发起者造成等于其攻击力的伤害
  4. 双方伤害同时结算
  5. 不受嘲讽影响（可越过嘲讽单挑后排）
  6. 不受冰冻影响（被冰冻的单位被单挑时仍可反击）
  7. 单挑后，发起者本回合不可再攻击（类似已攻击状态）
```

**设计意图**：
- 提供绕过嘲讽的战术选择
- 模拟三国演义中武将单挑的名场面
- 高风险：你的武将可能被反杀

**使用场景**：卡牌效果——「威名：对一个敌方武将发起单挑」

---

#### 【新关键词 2】羁绊（Bond）

**定义**：当你场上同时存在指定的多个武将时，触发羁绊效果。

**规则细节**：
```
羁绊触发条件：
  - 羁绊的参与武将全部在你的场上
  - 最后一个羁绊武将入场时触发（类似条件满足时的一次性效果）

羁绊效果类型（不同组合不同效果）：
  - 「桃园结义」（刘备 + 关羽 + 张飞）：对所有友方武将造成 1 点伤害，然后使其获得 +2/+2
  - 「五虎上将」（关羽 + 张飞 + 赵云 + 马超 + 黄忠）：额外触发全体友军 +3/+3 和铁壁
  - 「赤壁之战」（周瑜 + 黄盖 + 诸葛亮）：对敌方所有武将造成 3 点伤害
  - 「虎豹骑」（夏侯惇 + 夏侯渊 + 曹仁）：所有魏国武将获得突阵（冲锋）
```

**程序实现**：
```
羁绊是一个条件触发的效果：
  - 每当一个武将入场时，检查所有羁绊组合
  - 如果某个羁绊组合的所有成员都在场上 → 触发羁绊效果
  - 每个羁绊组合每局游戏只能触发一次
```

**设计意图**：
- 鼓励收集和搭配特定武将组合
- 提供构筑层面的深度
- 三国名组合是玩家最熟悉的内容，代入感极强

---

#### 【新关键词 3】士气（Morale）

**定义**：士气是英雄的额外生命值（等同于炉石的护甲机制），但额外附带：当己方士气 ≥ 5 时触发「士气高涨」效果；当己方士气 ≤ 2 时触发「士气低落」效果。

**规则细节**：
```
士气 = 护甲值（规则完全等同于炉石护甲）

额外规则：
  士气 ≥ 5 时 —— 你的英雄获得"士气高涨"：
    所有友方武将获得 +1 攻击力（光环效果）
  
  士气 ≤ 2 时 —— 你的英雄获得"士气低落"：
    所有友方武将获得 -1 攻击力（光环效果）
  
  士气在 3-4 之间时：无光环效果

  注意：士气光环是持续效果，实时更新（士气增减时立即更新）
       士气光环不叠加——多个士气来源不重复触发
```

**设计意图**：
- 让护甲（士气）不再只是「纯防御属性」，而成为一个有策略深度的资源
- 高士气时推进进攻，低士气时被迫防守
- 模拟古代战争中士气对战斗力的影响

---

#### 【新关键词 4】火攻（Fire Attack）

**定义**：火攻计策或武将造成的伤害额外附带「灼烧」效果——目标在下个回合开始时受到 1 点伤害。

**规则细节**：
```
火攻造成伤害后：
  1. 正常结算当前伤害
  2. 在目标身上附加一个「灼烧」debuff 标记
  3. 灼烧效果：
     - 下个回合开始时，目标受到 1 点伤害
     - 灼烧只触发一次，触发后消失
     - 灼烧不可叠加（多次火攻不累加伤害，只刷新下回合触发）
     - 灼烧可被离间（沉默）移除
  4. 灼烧对英雄也生效（下回合开始时扣血）

火攻与非火攻的区别：
  - 普通伤害：只在当前结算
  - 火攻伤害：当前结算 + 下回合开始时额外 1 点伤害
```

**设计意图**：
- 三国著名计策（火烧赤壁、火烧连营）的具象化
- 提供延迟伤害的战术维度
- 与激愤（暴怒）等关键词产生有趣的对抗

## 五、三国专属新机制

### 5.1 兵种系统

替代炉石的种族（Tribes）系统。每个武将属于一个兵种：

| 兵种 | 风格 | 示例武将 |
|------|------|----------|
| **步兵** | 均衡属性，适合铺场 | 廖化、周仓、潘凤 |
| **骑兵** | 高攻击力，低生命值 | 关羽（赤兔马）、吕布（赤兔马）、赵云 |
| **弓兵** | 远程输出，不参与直接战斗（特殊规则见下） | 黄忠、太史慈、夏侯渊 |
| **谋士** | 低攻击，强法术效果 | 诸葛亮、司马懿、周瑜、郭嘉 |
| **方士** | 治疗、buff、特殊效果 | 华佗、于吉、左慈、张角 |

**兵种协同机制**：
- 某些计策只对特定兵种生效（如「万箭齐发」只对弓兵生效）
- 某些武将效果与兵种相关（如「夏侯惇：友方骑兵获得 +1 攻击力」）

### 5.2 攻城与守城模式（可选扩展）

此为**后续版本**的扩展内容，第一期可不做。
- **攻城方**：额外获得攻击力加成
- **守城方**：所有随从获得 +1 生命值
- 某些计策牌可以改变攻守态势

## 六、战场规则调整

### 6.1 核心规则：完全保留

以下战场规则**不做任何改动**：
- 双方各 7 个随从站位上限
- 随从按打出顺序从左到右排列
- 站位固定不自动填补
- 相邻效果存在
- 坟场机制完全保留

### 6.2 新增：武将展示

- 场上每个武将显示其：

## 九、卡牌设计原则

### 9.1 基础原则

1. **尊重历史 + 游戏性优先**：卡牌效果优先考虑平衡性和趣味性，对历史典故做合理的游戏化改编
2. **简单易懂**：每张卡牌文本不超过 3 行，复杂效果留给高稀有度卡牌
3. **阵营特色鲜明**：蜀善 buff、魏善控制、吴善过牌、群雄善进攻
4. **第一版保守，后续拓展**：第一期卡牌不做过于复杂的效果

### 9.2 稀有度设计

| 稀有度 | 名称 | 特点 | 牌组数量限制 |
|:---:|------|------|:---:|
| 普通 | 士卒 | 基础属性，无效果或简单效果 | 2 |
| 稀有 | 将领 | 有 1 个中等强度效果 | 2 |
| 史诗 | 名将 | 有独特机制效果 | 2 |
| 传说 | 无双 | 极强的独特效果，同名限 1 张 | 1 |

### 9.3 中立 vs 阵营专属

- **中立武将/计策**：所有阵营均可使用（如潘凤、华雄、袁术等非核心人物）
- **阵营专属**：蜀/魏/吴/群雄专属卡牌（如桃园结义仅蜀可用）
- 牌组中**必须**包含至少 10 张中立卡牌（？待讨论）
- 参考炉石：中立卡 + 职业卡混编
