---

## 0 · 立项背景

**目标**：做一款 PvE 卡牌对战 demo，最终封装成本地可下发的安装包给朋友玩。
**风格基调**：HoYoverse anime cel-shading（偏原神二次元），不要 painterly / DW9 半写实 / 工笔国画。
**核心玩法**：炉石传说式，但题材换三国 + 阵营对抗（蜀 / 吴）。
**目标平台**：浏览器（本地 dev + Vercel 公开版）+ 后续 ZIP 安装包。

## 跨周的关键技术决策

| 类别 | 决策 |
|:--|:--|
| 美术风格 | HoYoverse anime cel-shading（hard chiaroscuro + flat color + clean line + NO painterly/grain）|
| 卡牌视觉 | 三态系统：手牌 cardvisual + 场上 BattleCardVisual + 装备椭圆 clip-path |
| 阵营简化 | 三阵营 → 二阵营（蜀 + 吴 + 中立 7 张）|
| Mana 公式 | 1 mana ≈ 2.0-2.2 vanilla stats 基线 + 关键词税 |
| Class Identity | 优先于绝对平衡 — 蜀冲锋亏模、吴控制溢出、中立 by design 偏弱 |
| PvE UX 最高法则 | "让玩家爽" — 平局 → 玩家胜，AI 难度只一档 |
| 操作日志 | 严禁含敌方私有信息（防 DevTools 开挂 + PvP 数据隔离）|
