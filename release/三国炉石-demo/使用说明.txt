三国炉石 Demo · 启动说明
================================

【一键启动】

  双击  启动游戏.bat


【操作流程】

  1. 双击「启动游戏.bat」
  2. 弹出黑色命令窗口（请勿关闭）
  3. 浏览器自动打开 → 进入游戏


【系统要求】

  · Windows 7 / 10 / 11
  · 任意现代浏览器（Edge / Chrome / Firefox 均可）
  · 无需安装 Node.js / Python / 任何额外软件
  · 仅使用系统自带的 PowerShell


【退出游戏】

  关闭浏览器 → 关闭黑色命令窗口


【常见问题】

  Q: 浏览器没有自动打开？
  A: 手动访问 http://localhost:5173/

  Q: 提示"无法运行脚本"或"PowerShell 被禁用"？
  A: 你的电脑系统策略限制脚本运行，请联系作者获取离线版本

  Q: 启动后立即闪退？
  A: 端口 5173-5189 被占用。先关闭其他占用端口的程序再试。


【关于游戏】

  三国题材的卡牌对战 demo
  · 双击进入游戏
  · 跳过 / 看完开场动画
  · 勾选用户协议 → 进入游戏
  · 体验主菜单 / 卡牌图鉴 / 各功能子页面

  当前版本：W1 视觉打磨阶段 demo
  战场对战逻辑（W2-W4）尚未实装
